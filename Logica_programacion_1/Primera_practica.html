Podemos escribir texto sin la necesidad de usar otras etiquetas.
Sin embargo, es necesario agregar < > para poder visualizar acentuaciones dentro de otros navegadores como explorer.
En el caso de querer agregar operaciones matemáticas debemos usar java script, ya que HTML no puede hacerlas.
Para esto se necesita agregar < > para poder empezar a escribir en java script.
Si queremos escribir dentro del script es necesario agregar " document.write(" ") " poniendo dentro de los parentesis el TEXTO a escribir.
Si se quieren hacer operaciones en vez de agregar comillas dobles dentro del parentesis, se agregan las operaciones en sí.
Por ejemplo: 18+2 =

Se pueden concatenar elementos, ya sea cadena de caracteres y/o números, simplemente remplazando los valores dentro de los paréntesis.


De esta forma nos aceguramos que se hace la operación correcta y el resultado se concatena con la cadena de caracteres.


Recordar siempre que las etiquetas van en minusculas, aunque es posible ponerlas en mayúsculas también.
